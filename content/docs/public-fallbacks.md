# Public Fallbacks

This page shows all public fallbacks that out community have created

## Doughmination Restful

This api endpoint is a mix of both Lanyard and Dustin's API, maintained by Clove.

The original intent was for personal use, however fae does not mind users to use the endpoint as a fallback in case Dustin's API goes down.

The endpoint can be found at: https://restful.doughmination.uk/v1/users/{USER_ID}, documentation can be found on faer site, at [doughmination.uk](https://doughmination.uk/docs/restful) and source code at https://git.gay/doughmination/restful
